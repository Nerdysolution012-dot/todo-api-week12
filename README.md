# 📝 Todo API

A simple RESTful API built with ASP.NET Core for managing Todo items.  
This project demonstrates CRUD operations, error handling, and clean API design.

---

## 🚀 Features
- Create new Todo items
- Retrieve all Todos
- Retrieve a Todo by ID
- Update existing Todos
- Delete Todos
- Structured error responses

---

## ⚙️ Technologies
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server (or SQLite for local development)
- C#

---
